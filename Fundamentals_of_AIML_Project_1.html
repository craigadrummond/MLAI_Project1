Context

Cardio Good Fitness is seeking to identify differneces between the customers of three of their products. They would like a customer profile for each product with insights and recommendations that will improve marketing efforts

Objective

  • Identify the customer profile for each product
  • Identify recommendations for marketing each product

Key Questions

  • What is the age range for each product?
  • What is the income range from each product?
  • Which product should be marketed to with gender?
  • Which prodcut should be marketed to customers who consider themselve very fit?
  • Does Education or Income factor into the which product is purchased?

Executive Summary¶

Cardio Good Fitness is looking to expand it marketting efforts for their three different treadmill products. To effectively utilize the limited marketing budget customer profiles are needed to accomplish the company goals

To perform this analysis a dataset of 180 previous sales was analyed. In this data set the following was observed:

  • The minimum age of customers is 18. The maximum is 50. The average age is 28.7
  • The number of male customers is 104 and the number of female customers is 76
  • The number of married customers is 107 and the number of single cusomers is 73.
  • The overall average income of all customers is $53,719
  • The average number of years of education is 15.5 but education only showed minimal impact on product with the highest education correlating to income and higher income customers did tend to purchase the TM798

def histogram_boxplot(data, feature, figsize=(12, 7), kde=False, bins=None):
    """
    Boxplot and histogram combined

    data: dataframe
    feature: dataframe column
    figsize: size of figure (default (12,7))
    kde: whether to show the density curve (default False)
    bins: number of bins for histogram (default None)
    """
    f2, (ax_box2, ax_hist2) = plt.subplots(
        nrows=2,  # Number of rows of the subplot grid= 2
        sharex=True,  # x-axis will be shared among all subplots
        gridspec_kw={"height_ratios": (0.25, 0.75)},
        figsize=figsize)  # creating the 2 subplots
    
    sns.boxplot(data=data, x=feature, ax=ax_box2, showmeans=True, color="violet")  # boxplot will be created and a star will indicate the mean value of the column
    
    if bins:
      sns.histplot(data=data, x=feature, kde=kde, ax=ax_hist2, bins=bins, palette="winter")
    else: 
      sns.histplot(data=data, x=feature, kde=kde, ax=ax_hist2)  # For histogram
    
    ax_hist2.axvline(data[feature].mean(), color="green", linestyle="--")  # Add mean to the histogram
    
    ax_hist2.axvline(data[feature].median(), color="black", linestyle="-")  # Add median to the histogram

def labeled_barplot(data, x, y=None, hue=None, n=None):
    """
    Barplot with percentage at the top

    data: dataframe
    x: dataframe categorical variable
    y: dataframe numerical variable, 
      if y is None, use countplot, otherwis use barplot 
    hue: dataframe column for addition category
    n: displays the top n category levels (default is None, i.e., display all levels)
    """

    count = data[x].nunique()
    if hue is None:
        m = 1
    else:
        m = data[hue].nunique()
    
    if n is None:
        plt.figure(figsize=(count * m + 3, 7))
    else:
        plt.figure(figsize=((n + 1) * m + 3, 7))

    plt.xticks(rotation=90, fontsize=15)
    if y:
      ax = sns.barplot(
          data=data,
          x=x, 
          y=y, 
          hue=hue, 
          order=data.sort_values(y, ascending=False).head(n)[x],
          palette ='Pastel2'
          )
    else:
      ax = sns.countplot(
          data=data,
          x=x,
          hue=hue,
          order=data[x].value_counts().sort_values(ascending=False).index[:n],
          palette="Paired"
          )
    
    for c in ax.containers:
            ax.bar_label(c, fmt='%.2f')
